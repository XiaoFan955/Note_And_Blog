### 数学符号

#### 比较运算符

|   名称   | code  | 效果  |
| :------: | :---: | :---: |
|   小于   | `\lt` | $\lt$ |
|   大于   | `\gt` | $\gt$ |
| 小于等于 | `\le` | $\le$ |
| 大于等于 | `\ge` | $\ge$ |
|  不等于  | `\ne` | $\ne$ |

可以在这些运算符前面加上`\not` ，如`\not\lt` : $\not\lt$
